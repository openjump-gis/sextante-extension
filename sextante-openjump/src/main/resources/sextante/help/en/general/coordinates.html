The SEXTANTE pick coordinates

This tool allows to collect the coordinates of points on the view. Those points can be selected and used as input into an algorithm, thus avoiding to type them.

Point coordinates are captured by selecting the tool and clicking on the view. SEXTANTE informs the use with a simple message where it is possible to enter a name for the point.

Once user executes the dialog of an algorithm, he has only to click the button that appears on the left side of the field to select the collected coordinates of the point
