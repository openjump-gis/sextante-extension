SEXTANTE and OpenJUMP

This section describes how to load data into Sextante in OpenJUMP

Vector Layers:

Those layers can be load using the option "File>Open" and choose one of the three different options: a) File, b) Data Store Layer, c) Writable Data Store


WFS vector layers can be used into Sextante. Image or raster layers loaded using the "Open>File" option will be ignored by Sextante

Raster Layers:

Those layers can be load using the option "File>Open" and choose the option "Sextante Raster Layer"


Tables:

Currently the only way to load table files (like dbf or Excell) into OpenJUMP is to save those table to .csv text format and to load into OpenJUMP using the option "File>Open" and choose the next option "CSV (Auto)"
